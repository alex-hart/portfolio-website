About Me

Hi! I'm Alex Hart, a 24-year old engineering graduate currently living in Chatham, UK. I am an aspiring front-end web developer with a passion for all things web-related and a drive to continually improve my developing skills.

I stumbled across web development during my time at University and since graduating I have dedicated my time to learning as much as I can by using all resources at my disposal. I am now actively looking for full-time web development job opportunities which will allow me to further learn and grow as a junior front-end web developer while also being able to put my current coding skills to the test.

I am familiar and comfortable with HTML, CSS, Sass and JavaScript as well as the Bootstrap and jQuery libraries, all of which I have used in my projects alongside Git and GitHub as version control. I am also in the process of teaching myself how to understand and use Angular and React which I look forward to implementing in my future work. In the near future I would like to start learning more languages and libaries - particularly those involved with back-end development such as PHP, node.js and mongoDB.
